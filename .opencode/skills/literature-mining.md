# Literature Mining Skill

Use Semantic Scholar to find materials for insulin delivery, transdermal patches, and protein stabilization.

## Typical Queries

- Hydrogels insulin stabilization transdermal
- Biocompatible polymers drug delivery skin
- Protein stabilization thermal polymers
- Smart materials temperature-responsive release

## Output

Material candidates with: name, composition, thermal stability, biocompatibility, literature references. Saved to `iterative_results/` and `mining_results/`.

## MCP Tool

`mine_literature(query, max_candidates)` – Run via MCP insulin-ai server.
